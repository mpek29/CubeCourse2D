Ceci est le manuel d'utilisation du jeu Cube Course 2D


1 ) Lancer le jeu :
	- Ouvrir un terminal dans le dossier Jeu
 	- Tapez la commande suivante : python main.py
 	- Jouez


2 ) Objectif du jeu : 
	Le but du jeu Cube Course 2D est de survivre dans un monde peupler d'enemies et de platforme en mouvement le plus longtemps possible. Il vous faudra donc sauter de platforme en platforme et intéragir éviter les différents 
	enemies. De plus, il faudra tenter de réaliser le meilleur temps possible car ce jeu est une course comme sont nom l'indique. Pour cela, un classemebnt est disponible en fin de partie.
	

3 ) Les différentes touches : 
	[SPACE] -> Sauter
	[ECHAP] -> Quitter le jeu
	[BACKSPACE] -> Supprimer un caractère du nom dans l'accueil
	[Autres touches du clavier] -> Ecrire son pseudo dans l'accueil
